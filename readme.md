# Manga Trans 가이드

## 이게 뭔가요?
일본 만화 이미지를 한국어로 번역해주는 뷰어 프로그램입니다.  
이미지를 열고 스페이스바만 누르면 번역된 텍스트가 그림 위에 바로 표시됩니다.

---

## 시작 전 필수 준비: 일본어 언어팩 설치

Windows에 기본 내장된 OCR 기능을 쓰기 때문에, 일본어 언어팩이 없으면 아예 작동하지 않습니다.

**설정 → 시간 및 언어 → 언어 → 언어 추가 → 일본어 선택**

이것만 하면 기본 기능은 바로 쓸 수 있습니다.

---

## 번역 품질을 높이고 싶다면: Manga-OCR 설치

기본 Windows OCR은 만화 글씨 인식이 좀 아쉽습니다. 만화 전용 AI 모델인 **Manga-OCR**을 쓰면 훨씬 잘 됩니다.

**설정 → 번역 엔진 설정 → Manga OCR 선택** 후 안내에 따라 설치하면 됩니다.

단, 설치에 시간이 꽤 걸립니다.
- 다운로드 용량: **3GB 이상**
- 필요 여유 공간: **C드라이브 10GB** (압축 해제와 복사를 위해 넉넉잡은 용량이고, 실제는 3GB 내외입니다)
- 소요 시간: 인터넷 속도에 따라 **최대 30분**

Python 3도 미리 설치되어 있어야 합니다.

---

## 번역 사용법

1. 이미지를 엽니다
2. **스페이스바**를 누릅니다
3. 끝입니다

번역은 기본적으로 꺼져 있습니다. 이미지 뷰어로만 쓸 때 불필요하게 CPU/메모리를 잡아먹지 않게 하기 위해서입니다.

Manga-OCR을 처음 실행할 때는 AI 모델을 불러오는 데 **1분 정도** 걸리지만, 그 이후 각 페이지 번역은 빠르게 됩니다.

---

## 지원 번역기

| 번역기 | 특징 |
|--------|------|
| Windows OCR | 기본 내장, 별도 설치 불필요 |
| Manga-OCR | 만화 최적화, 고품질 (Python 필요) |
| 구글 번역 | 번역 품질 우수 |
| ezTrans | 로컬 번역 지원 |
