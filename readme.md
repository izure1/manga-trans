# Manga Trans

일본어 -> 한국어 이미지 번역 뷰어 프로그램

## 핵심 기능

1. OCR 기술을 활용하여 일본어 텍스트 추출 후 해당 위치에 자동으로 번역된 텍스트 오버레이
2. ezTrans, 구글 번역 지원
3. 파이썬 설치 후, Manga-ocr 사용 시 최고 품질의 만화 번역 가능
4. 이미지 회전, 슬라이드쇼 기능 지원 및 번역 기능 없이도 일반적인 이미지 뷰어 기능 제공

## 사용자 환경

Windows 10, Windows 11

## 가이드

기본적으로 Windows 자체에 내장되어있는 Windows OCR 기능을 활용합니다.
따라서 일본어 언어팩을 필수로 설치해야합니다.

### 일본어 언어팩 설치 방법

1. 설정 - 시간 및 언어 - 언어로 들어갑니다.
2. 기본 설정 언어에 **언어 추가** 버튼을 누르고, 일본어를 추가합니다.

이로써 기본적인 Manga Trans 기능을 사용할 수 있습니다.
다만 Windows OCR은 만화에 적합한 도구가 아니므로, 텍스트 추출 성능이 많이 아쉬운 경우가 많습니다.

이 경우, Manga-OCR 파이썬 기능을 사용하여 만화에 최적화된 텍스트 추출 기능을 활용할 수 있습니다.

### Manga-OCR 설치 방법

1. Manga Trans 앱 하단에 설정 - 번역 엔진 설정으로 들어갑니다.
2. Manga OCR 텍스트 엔진을 선택합니다.
3. 안내에 따라 설치를 진행합니다.

Manga-OCR은 Python 3 설치가 필요하며, 별도의 패키지와 ai모델을 다운받아야하기 때문에 오래 걸리는 작업입니다.
다운로드 속도와 컴퓨터 성능에 따라 30분 가까이 걸릴지도 모릅니다. 끈기를 가지고 기다려 봅시다.
